# SDL_HAPTIC_CUSTOM

Custom effect is supported.

## Header File

Defined in [SDL_haptic.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_haptic.h)

## Syntax

```c
#define SDL_HAPTIC_CUSTOM     (1u<<11)
```

## Remarks

User defined custom haptic effect.
