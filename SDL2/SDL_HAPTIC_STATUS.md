# SDL_HAPTIC_STATUS

Device can be queried for effect status.

## Header File

Defined in [SDL_haptic.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_haptic.h)

## Syntax

```c
#define SDL_HAPTIC_STATUS     (1u<<14)
```

## Remarks

Device supports querying effect status.
