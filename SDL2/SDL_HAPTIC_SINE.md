# SDL_HAPTIC_SINE

Sine wave effect supported.

## Header File

Defined in [SDL_haptic.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_haptic.h)

## Syntax

```c
#define SDL_HAPTIC_SINE       (1u<<1)
```

## Remarks

Periodic haptic effect that simulates sine waves.
