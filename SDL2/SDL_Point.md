# SDL_Point

The structure that defines a point (integer)

## Header File

Defined in [SDL_rect.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_rect.h)

## Syntax

```c
typedef struct SDL_Point
{
    int x;
    int y;
} SDL_Point;
```
