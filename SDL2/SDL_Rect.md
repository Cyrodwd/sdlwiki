# SDL_Rect

A rectangle, with the origin at the upper left (integer).

## Header File

Defined in [SDL_rect.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_rect.h)

## Syntax

```c
typedef struct SDL_Rect
{
    int x, y;
    int w, h;
} SDL_Rect;
```
