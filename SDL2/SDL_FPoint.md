# SDL_FPoint

The structure that defines a point (floating point)

## Header File

Defined in [SDL_rect.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_rect.h)

## Syntax

```c
typedef struct SDL_FPoint
{
    float x;
    float y;
} SDL_FPoint;
```
