# SDL_FreeAudioStream

Free an audio stream

## Header File

Defined in [SDL_audio.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_audio.h)

## Syntax

```c
void SDL_FreeAudioStream(SDL_AudioStream *stream);
```

## Version

This function is available since SDL 2.0.7.
