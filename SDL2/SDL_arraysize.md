# SDL_arraysize

The number of elements in an array.

## Header File

Defined in [SDL_stdinc.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_stdinc.h)

## Syntax

```c
#define SDL_arraysize(array)    (sizeof(array)/sizeof(array[0]))
```
