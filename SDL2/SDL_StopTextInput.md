# SDL_StopTextInput

Stop receiving any text input events.

## Header File

Defined in [SDL_keyboard.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_keyboard.h)

## Syntax

```c
void SDL_StopTextInput(void);
```

## Version

This function is available since SDL 2.0.0.
