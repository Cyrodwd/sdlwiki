# SDL_ALPHA_OPAQUE

A fully opaque 8-bit alpha value.

## Header File

Defined in [<SDL3/SDL_pixels.h>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_pixels.h)

## Syntax

```c
#define SDL_ALPHA_OPAQUE 255
```

## Version

This macro is available since SDL 3.1.3.
