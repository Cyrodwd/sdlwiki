# SDL_MS_PER_SECOND

Number of milliseconds in a second.

## Header File

Defined in [<SDL3/SDL_timer.h>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_timer.h)

## Syntax

```c
#define SDL_MS_PER_SECOND   1000
```

## Remarks

This is always 1000.

## Version

This macro is available since SDL 3.2.0.
