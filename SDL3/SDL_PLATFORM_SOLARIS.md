# SDL_PLATFORM_SOLARIS

A preprocessor macro that is only defined if compiling for SunOS/Solaris.

## Header File

Defined in [<SDL3/SDL_platform_defines.h>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_platform_defines.h)

## Syntax

```c
#define SDL_PLATFORM_SOLARIS 1
```

## Version

This macro is available since SDL 3.2.0.
